This program reads student module grades from text files
and outputs transcript information for each student.

## Build Instructions (VS Code)

Press CTRL + SHIFT + B

This runs the build task defined in `.vscode/tasks.json`,
which compiles:

- main.cpp
- classes/Student.cpp
- classes/Term.cpp

and produces the executable:
`program.exe`

## How to Run

Open the VS Code terminal in the project root and run:
`./program.exe`
